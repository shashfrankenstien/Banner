# Banner

Converts string to a banner that moves on the console from right to left

```
$ python moving_banner.py -h

usage: python moving_banner.py [-h] [-c C] [-x X] [-y Y] [--width WIDTH] [--speed SPEED] [--quiet] message

positional arguments:
  message               message to print as banner

optional arguments:
  -h, --help            show this help message and exit
  -c C                  character used for banner text
  -x X                  horizontal wall character
  -y Y                  vertical wall character
  --width WIDTH, -w WIDTH
                        width of the banner
  --speed SPEED, -s SPEED
                        speed of the banner
  --quiet, -q           don't print extra text

```
